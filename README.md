# Edinburgh Arts Info

A simple static website for EdinburghArts.info, focused on galleries, exhibitions, art routes and festival-season navigation.

## What it does

- Presents Edinburgh Arts Info as an independent visual art guide
- Lists exhibitions, galleries, openings and art spaces
- Lets visitors filter by view, area, search term and quick tags
- Includes easy art routes for locals, tourists and festival visitors
- Adds source links and Google Calendar links for listings
- Works as a static site suitable for GitHub Pages

## Files

- `index.html`, page structure and SEO metadata
- `styles.css`, responsive visual design
- `app.js`, sample listings, filters, routes and calendar links

## Positioning

Edinburgh Arts Info is an independent guide to what is showing in Edinburgh galleries today. It is designed to be clearer, calmer and more useful than a scattered set of venue pages or overloaded festival listings.
